# ShiHuaMusic

ShiHua音乐播放器是一个基于HarmonyOS的音乐播放应用。

## 项目介绍

这是一个使用DevEco Studio开发的HarmonyOS音乐播放器应用，支持本地音乐播放功能。

## 功能特性

- 本地音乐文件播放
- 音乐列表展示
- 播放控制（播放、暂停、下一首、上一首等）
- 音量控制

## 开发环境

- DevEco Studio 6.0.0.868
- HarmonyOS SDK
- Windows 24H2

## 安装说明

1. 克隆或下载本项目源代码
2. 使用DevEco Studio打开项目
3. 构建并运行到设备或模拟器

## 使用方法

1. 应用启动后会自动扫描设备中的音乐文件
2. 在主界面可以看到音乐列表
3. 点击任意音乐即可开始播放
